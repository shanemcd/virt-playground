# Agent Orientation

## What this repo is

A public learning journal for OpenShift Virtualization. Contains architecture docs and hands-on labs. No code to build or test. All content is markdown.

**This repo is public.** No personal or internal information.

## Repo structure

- `docs/` - Architecture and reference documentation
- `labs/` - Hands-on lab sequences, each in its own directory with a README and numbered steps
- `docs/ecosystem-map.md` - Map of all projects/repos in the virt ecosystem

## Writing style

- Technical, concise, no marketing language
- Labs follow a pattern: README with table + architecture diagram, numbered step files
- Docs cover mechanics ("how does X actually work"), not just usage
